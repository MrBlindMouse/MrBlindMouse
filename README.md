### [Roomz](https://github.com/MrBlindMouse/Roomz)
This is a music streaming server, mimicing a Radio channel. A global music player is streamed in sync to all devices.

The purpose is to have multiple devices play the same track, synced throughout a building. I started this because I was annoyed that I cannot connect more than one BT speaker to the same phone.
